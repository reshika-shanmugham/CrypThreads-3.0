## ⚡ 5-Minute Setup

### Step 1: Register (30 seconds)
1. Click **"Register"** link
2. Enter username (min 3 characters)
3. Enter password (min 6 characters)
4. Confirm password
5. Click **"Register"**

### Step 2: Login (15 seconds)
1. Enter your username
2. Enter your password
3. Click **"Login"**
4. ✅ You're in! Session lasts 30 days

### Step 3: Add Another User (30 seconds)
1. Click **"Add New User"** (right sidebar)
2. Enter username: `friend`
3. Enter password: `password123`
4. Click **"Add User"**

### Step 4: Send a Message (45 seconds)
1. Select **"friend"** from recipient dropdown
2. Type: `Hello! Welcome to CrypThreads!`
3. Click **"Send Message"**
4. ✅ Message added to pending queue

### Step 5: Mine Messages (30 seconds)
1. Click **"Mine Pending Messages"** (right sidebar)
2. Wait 2-3 seconds for mining
3. ✅ Message delivered!

### Step 6: View Results (30 seconds)
1. Check statistics (left sidebar):
   - Total Blocks: 2
   - Pending Messages: 0
   - Total Messages: 1
2. See your message in the list
3. Status shows: ✓ Mined & Delivered

## 💡 Pro Tips

### Messaging
- **Quick Send**: Press `Ctrl+Enter` in message box
- **Batch Mining**: Send multiple messages, then mine once
- **Check Status**: Look for ⏳ (pending) or ✓ (mined)

### Session
- **Stay Logged In**: Session lasts 30 days automatically
- **Multiple Devices**: Login from any device with same credentials
- **No Re-registration**: Once registered, always registered

### User Management
- **Add Users Anytime**: No limit on number of users
- **Delete Users**: Remove inactive users to keep list clean
- **Cannot Delete Self**: Must logout first

## 📊 Understanding the Dashboard

### Left Sidebar (Statistics)
```
🔗 Total Blocks: Number of blockchain blocks
⏳ Pending Messages: Messages waiting to mine
📨 Total Messages: Your received messages
👥 Total Users: Registered users
✅ Blockchain Status: Always "Valid & Verified"
```

### Main Area (Messages)
```
From/To: Shows sender or recipient
Timestamp: When message was sent
Content: Your message text
Status: Pending or Mined
```

### Right Sidebar (Actions)
```
⛏️ Mine: Process pending messages
👥 View Users: See all registered users
➕ Add User: Register new user
🚪 Logout: End session
```

## ⚠️ Important Notes

### Data Storage
- All data stored in **browser localStorage**
- Clearing browser data = losing everything
- Use same browser to maintain data

### Sessions
- **30-day validity** from last login
- Works across devices
- Auto-expires after 30 days

### Mining
- **Required** for message delivery
- Takes 2-3 seconds
- Creates permanent blockchain record

### Security
- Passwords are hashed
- Messages are encrypted
- Sessions are persistent

### Button Types
- **Cyan Gradient**: Primary actions (Login, Register, Mine)
- **Dark with Border**: Secondary actions (View, Add)
